# Etch-a-Sketch

A browser-based drawing application built using **HTML, CSS, and vanilla JavaScript** as part of  
**The Odin Project – JavaScript Fundamentals**.

The project focuses on DOM manipulation, event handling, and dynamically generating grid elements.

---

## Features

- Draw by hovering over grid cells
- Adjustable grid size
- Reset / clear functionality
- Dynamic grid generation using JavaScript
- Responsive layout

---

## What I Learned

- Creating and modifying DOM elements dynamically
- Handling user interactions with event listeners
- Managing application state for grid size and resets
- Using CSS Grid for layout
- Structuring JavaScript logic for interactive UI behavior

---

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)

No frameworks or external libraries were used.

---

## Project Context

This project was built as part of **The Odin Project – JavaScript Fundamentals** curriculum.

The goal of the assignment was to strengthen understanding of DOM manipulation and user interaction without relying on external abstractions.
